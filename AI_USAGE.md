# AI Usage Documentation

## Tools Used

For this project, I used ChatGPT as a support tool during development.

## How I Used AI

I mainly used AI to help with:

- checking parts of my Flask code
- fixing small errors
- getting help with HTML page structure
- understanding how to connect the database to the app
- improving the organization of some files

## Example Prompts I Used

Some prompts I used were:

- "Help me fix this Flask route."
- "How do I connect Flask to MySQL?"
- "What should go in base.html for this project?"
- "How can I create a job match feature using skills?"
- "Help me write setup instructions for my README."

## What AI Helped With

AI was most helpful when I needed a starting point or when I got stuck on an error. It helped me understand how different parts of the project fit together, especially the Flask routes, templates, and database connection.

## What I Changed

I did not copy everything directly. I reviewed the suggestions and changed parts to fit my project.

Some changes I made were:

- adjusted code to match my table names and route names
- changed parts of the HTML templates to fit my project pages
- fixed errors when variables or template names did not match
- simplified some sections so they were easier for me to understand
- edited the README to match the requirements of the project

## What I Learned

Using AI helped me when I was stuck, but I still had to test everything and make corrections myself. I learned that even small mistakes in route names, template names, or variables can cause the app not to work. I also learned that AI is more helpful when I use it to guide me and explain things rather than doing the whole project for me.

## Reflection

Overall, AI was useful as a support tool for brainstorming, debugging, and checking my work. I still had to go through the code, make edits, and make sure everything matched the project instructions.
